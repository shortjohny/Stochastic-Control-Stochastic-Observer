Ονοματολογία:
P:Position V:Velocity A:Acceleration

Υπάρχουν 4 φάκελοι, για κάθε περίπτωση. Κάθε φάκελος περιέχει 4 υποφάκελους για κάθε περίπτωση διασποράς θορύβου μετρήσεων. Σε κάθε υποφάκελο υπάρχει ένας φάκελος για τα αποτελέσματα του φίλτρου Kalman, ένας φάκελος για τα αποτελέσματα του Στοχαστικού παρατηρητή και ένας φάκελος για τις εικόνες των simulation. Ακολουθείται η παραπάνω ονοματολογία.
